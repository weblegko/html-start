# Стартовый шаблон чтоб быстро начать пилить html  
> Сборка Gulp + Webpack  

Перейти в папку с исходниками для начала разработки  
`cd _dev`

Установить все зависимости  
`npm i`

Собрать и работать над проектом  
`gulp`

Собрать production  
`gulp build --prod`

Удалить собранный проект  
`gulp clear`
